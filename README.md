# Express on Bun

Пример приложения [Express](https://expressjs.com/) на рантайме [Bun](https://bun.sh/), которое можно развернуть в **Timeweb Cloud Apps** без настройки.

:tada: Демо

:rocket: [Создать свой Apps](https://timeweb.cloud/my/apps/create)

:books: [Документация Timeweb Cloud Apps](https://timeweb.cloud/docs/apps)

## <a name="dev"></a>Локальный запуск проекта

```bash
# установка зависимостей
bun install

# запуск приложения
bun run start
```
